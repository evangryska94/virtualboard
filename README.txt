VirtualBoard is an online platform that lets you connect with your friends in
a virtual tabletop environment.  Simply click the button above, pick a name
and color, and start playing!

Installation
1. pip install tornado
2. pip install jsonschema

Skip this step if you do not care about tests
3. pip install selenium

4. cd src
5. python ./launch.py
6. connect to http://localhost:8000
